# Retail Analytics Dashboard

## Project Overview
The Retail Analytics Dashboard is a comprehensive tool designed to visualize and analyze retail performance data. This project provides insights into sales trends, inventory levels, and customer demographics, enabling better decision-making. 

## Features
- Data Visualization
- Sales Performance Analysis
- Inventory Management
- Customer Insights

## Technologies Used
- Python
- Flask
- Pandas
- Plotly
- SQLAlchemy

## Installation
To install the project, clone the repository and install the necessary dependencies:
```bash
git clone https://github.com/chefmaahi/retail-analytics-dashboard.git
cd retail-analytics-dashboard
pip install -r requirements.txt
```

## Usage
Run the application using:
```bash
python app.py
```
Access the dashboard at `http://127.0.0.1:5000`
